# Number Guessing Game 🎯

A simple Python console game that challenges users to guess a secret number within a limited number of attempts.

## Features

- Uses Python while loops
- Takes user input from the keyboard
- Tracks the number of attempts
- Limits the player to 3 guesses
- Displays win or lose messages

## How It Works

1. The program stores a secret number.
2. The user is given 3 chances to guess it.
3. If the user guesses correctly, they win.
4. If all attempts are used, the game ends with a failure message.

## Technologies Used

- Python
- While Loops
- Conditional Statements
- User Input

## Learning Objectives

- Understand while loops
- Practice conditional logic (if, else)
- Work with variables and counters
- Handle user input
